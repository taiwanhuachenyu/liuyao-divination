# 六爻排盘

一个精美的在线六爻排盘工具，传统中式古风界面。

## 功能特性

- **三种起卦方式**
  - 铜钱摇卦：模拟三枚铜钱抛掷六次
  - 手动选卦：逐爻选择阴阳动变
  - 天机起卦：梅花易数时间起卦

- **完整排盘信息**
  - 本卦、变卦展示
  - 纳甲、六亲、六神
  - 世爻、应爻定位
  - 动爻标记（老阳○、老阴×）
  - 卦辞、爻辞展示
  - 卦象解析断语

- **界面特色**
  - 新中式古风美学，宣纸纹理背景
  - 四种字体自由切换（楷体、隶书、宋体、黑体）
  - 流畅动画效果（铜钱翻转、爻象渐入、脉冲提示）
  - 历史记录本地保存
  - 响应式设计，支持手机平板

## 技术栈

- React 18 + TypeScript
- Vite
- TailwindCSS
- Zustand (状态管理)
- React Router
- Lucide React (图标)

## 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 在线使用

打开浏览器访问 `http://localhost:5173/` 即可使用。
